# System Architecture

```
React
   │
REST API
   │
Express
   │
Controllers
   │
Services
   │
Repositories
   │
Prisma ORM
   │
PostgreSQL

           AI Provider
               │
      Mock / Ollama / OpenAI
```

## Core Principles

- Modular architecture
- Repository pattern
- Service layer
- Dependency inversion for AI providers
- Stateless JWT authentication
- Background processing for long-running tasks
- Docker-first deployment
